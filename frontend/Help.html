Getting started

Agentive is ready to use from the moment you create your account. Follow these steps to send your first message.

1
Create your account

Go to the sign-up page and enter your name, email, and a password. Your workspace is provisioned immediately — no email verification delay.

2
Open the dashboard

After signing in, you land on the Dashboard. You'll see Platform Agents — pre-configured agents maintained by Agentive — ready to use immediately.

3
Click "Chat" on any agent

This opens the Chat page with that agent selected. A new conversation is created automatically when you send your first message.

4
Type and send

Type in the input bar and press Enter to send. Use Shift + Enter for a newline. The agent's response streams in word by word.

Creating custom agents

Custom agents let you build a specialised AI with a specific role, tone, and expertise — yours alone, not shared with other users.

1
Click "+ New Agent" on the dashboard

This opens the agent creation form.

2
Fill in the name and description

The name appears on your dashboard card. The description is shown as a subtitle — keep it concise and descriptive.

3
Write the system prompt

This is the most important field. It defines who your agent is. See the system prompt guide below for best practices.

4
Select a model and save

Choose the model that fits your use case (see the model guide below), then click Create Agent. Your agent appears on the dashboard immediately.

Tip: You can delete custom agents from the dashboard card using the delete button. Platform agents cannot be deleted — they are maintained by Agentive.

Chatting with agents

The chat interface is designed for extended, context-rich conversations.

→
Enter sends, Shift+Enter adds a newline

For multi-paragraph messages, use Shift+Enter to format before sending.

→
Copy any message

Hover over any message to reveal the Copy button. Code blocks also have individual copy buttons in their header bar.

→
Responses render as formatted text

Bold, lists, headings, tables, and code blocks all render properly. You don't need to ask the agent to avoid markdown — it renders it.

Managing conversations

→
Start a new conversation

Click + New in the sidebar. Each conversation maintains its own context — perfect for separating different projects or topics.

→
Search conversations

Use the search box in the sidebar to filter conversations by title. Titles are auto-generated from your first message.

→
Delete a conversation

Hover over any sidebar item and click the ✕ button. A confirmation dialog appears before deletion — this action is permanent.

Writing effective system prompts

The system prompt is the single most important configuration for your agent. Here is how to write one that works.

→
Define the role clearly

Start with "You are a [role] that specialises in [domain]." Be specific — vague prompts produce vague agents.

→
Set the tone

Tell the agent how to communicate: "Respond in a concise, professional tone" or "Use casual, friendly language and avoid jargon."

→
Define boundaries

Specify what the agent should and should not do: "Only answer questions related to Python. For other topics, politely redirect the user."

You are a senior Python engineer with 10+ years of experience. You specialise in writing clean, well-documented, production-grade code. Always explain your reasoning before showing code. Format all code in proper Python with type hints. If the user's question is unclear, ask for clarification before answering.
Tip: Longer prompts are not always better. A focused 100-word prompt outperforms a vague 500-word one. Test iteratively — send a few messages and refine the prompt based on the agent's actual responses.

Frequently asked questions

Why does the agent sometimes forget earlier messages?+
Every model has a context window — a maximum number of tokens it can process at once. On very long conversations, older messages are automatically trimmed from the context to keep the most recent exchange within the limit. Gemma 2 has the smallest window (8k), so this happens sooner with that model.
Can I have multiple conversations with the same agent?+
Yes. Click "+ New" in the sidebar to open a fresh conversation with the currently selected agent. Each conversation is independent with its own context. You can switch between them freely from the sidebar.
Can other users see my conversations or custom agents?+
No. Conversations and custom agents are private to your account. Only platform agents are visible to all users. Every API request enforces ownership — you cannot access another user's resources, even if you know the ID.
What happens if the API returns an error mid-conversation?+
A toast notification appears describing the error. Your conversation history is preserved — the failed turn is not saved. You can retry by sending the same message again.
How do I delete my account?+
Account deletion is not yet self-serve. Please email us at devnexus26@gmail.com with your registered email address and we will process the deletion within 48 hours.
Is there a limit on how many messages I can send?+
The platform applies a rate limit of 20 messages per minute per IP to prevent abuse. For normal usage this limit will never be reached. If you do hit it, simply wait a minute and continue.
